# 0.3.11 First-Run Product Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make first-run CLI output clearer without changing JSON/SARIF schemas or exit-code behavior.

**Architecture:** Add small text-only guidance in reporters and tests. Keep machine-readable output stable. Coverage findings remain findings, while text output adds concise next steps for empty or weakly covered scans.

**Tech Stack:** TypeScript, Vitest, existing text reporters and CLI tests.
